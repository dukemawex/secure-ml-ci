# Secure ML CI

Security-focused continuous integration tooling for ML projects.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m secure_ml_ci
```

## Project structure

```
secure_ml_ci/        # source package
tests/      # tests
requirements.txt
```
